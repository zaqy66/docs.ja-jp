<span data-ttu-id="0d593-104">.NET Core は、NuGet パッケージで作成されたプラットフォームです。</span><span class="sxs-lookup"><span data-stu-id="0d593-104">.NET Core is a platform made of NuGet packages.</span></span> <span data-ttu-id="0d593-105">製品のエクスペリエンスには、粒度の細かいパッケージの定義から恩恵を受けるものもあれば、粒度の粗いパッケージの定義から恩恵を受けるものもあります。</span><span class="sxs-lookup"><span data-stu-id="0d593-105">Some product experiences benefit from fine-grained definition of packages while others from coarse-grained.</span></span> <span data-ttu-id="0d593-106">この二重性に対応するために、製品は、粒度の細かいパッケージ セットとして配布され、かつ、俗に[メタパッケージ](#metapackages)と呼ばれるパッケージの種類を使用して粒度の粗いチャンク単位で配布されます。</span><span class="sxs-lookup"><span data-stu-id="0d593-106">To accommodate this duality, the product is distributed as a fine-grained set of packages and in coarser chunks with a package type informally called a [metapackage](#metapackages).</span></span>

<span data-ttu-id="0d593-112">.NET core は、プリミティブ、上位レベルのデータ型、アプリ コンポジション型、および一般的なユーティリティを提供する一連のパッケージに分割されます。</span><span class="sxs-lookup"><span data-stu-id="0d593-112">.NET Core is split into a set of packages, which provide primitives, higher-level data types, app composition types and common utilities.</span></span> <span data-ttu-id="0d593-113">これらのパッケージはいずれも、同じ名前の単一アセンブリです。</span><span class="sxs-lookup"><span data-stu-id="0d593-113">Each of these packages represent a single assembly of the same name.</span></span> <span data-ttu-id="0d593-114">たとえば、[System.Runtime](https://www.nuget.org/packages/System.Runtime) には、System.Runtime.dll が含まれています。</span><span class="sxs-lookup"><span data-stu-id="0d593-114">For example, [System.Runtime](https://www.nuget.org/packages/System.Runtime) contains System.Runtime.dll.</span></span>

- <span data-ttu-id="0d593-116">粒度の細かいパッケージの場合は、ほかのパッケージのテストが比較的に抑えられるので、独自のスケジュールに合わせて配布することができます。</span><span class="sxs-lookup"><span data-stu-id="0d593-116">Fine-grained packages can ship on their own schedule with relatively limited testing of other packages.</span></span>

## <a name="package-based-frameworks"></a><span data-ttu-id="0d593-175">パッケージ ベースのフレームワーク</span><span class="sxs-lookup"><span data-stu-id="0d593-175">Package-based frameworks</span></span>
