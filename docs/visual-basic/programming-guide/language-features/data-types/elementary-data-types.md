# <a name="elementary-data-types-visual-basic"></a>基本データ型 (Visual Basic)
Visual Basic では、多くのプログラミング要素に使用できる定義済みのデータ型のセットが用意されています。 このセクションでは、これらの型とその使用方法について説明します。  
  
> [!NOTE]
>  Visual Basic では、各基本データ型が構造体またはクラス内にあるでサポートされている、<xref:System>名前空間。 コンパイラは、基になる構造体またはクラスのエイリアスとして各データ型のキーワードを使用します。 たとえば、予約語を使用して変数の宣言`Byte`は構造体の完全修飾名を使用して宣言することと同じ<xref:System.Byte?displayProperty=nameWithType>。
