<span data-ttu-id="d6507-103">プロパティ名は、匿名型の宣言で 1 回だけ確立できます。</span><span class="sxs-lookup"><span data-stu-id="d6507-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="d6507-104">たとえば、次の宣言は無効です。</span><span class="sxs-lookup"><span data-stu-id="d6507-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```
