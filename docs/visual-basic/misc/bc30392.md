# <a name="catch-cannot-catch-type-lttypenamegt-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>'Catch' で型をキャッチできません '&lt;typename&gt;' 'system.exception'、または 'system.exception' から継承するクラスではないため、
`Catch` は例外のみをインターセプトできますが、例外から派生していないものを検出しようとしました。  
  
 **エラー ID:** BC30392  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Catch` ステートメントを削除するか、または `Catch` の対象を実際の例外に変更します。
