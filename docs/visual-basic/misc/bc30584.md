# <a name="lttypegt-cannot-be-inherited-more-than-once"></a>'&lt;型&gt;' 2 回以上継承することはできません
`Inherits` ステートメントが重複しています。  
  
 **エラー ID:** BC30584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要な `Inherits` ステートメントを削除します。
