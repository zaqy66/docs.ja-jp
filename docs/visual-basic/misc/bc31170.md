# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>文字 'character' (0 x\<16 進数 >) は、XML 名で許可されていません
サポートされていない文字が、XML リテラルまたは XML 軸プロパティの名前に含まれています。  
  
 **エラー ID:** BC31170  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルまたは XML 軸プロパティの名前から、サポートされていない文字を削除します。
