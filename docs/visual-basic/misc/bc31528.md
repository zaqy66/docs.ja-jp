# <a name="construct-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-lttypenamegt"></a>コンス トラクターは、アセンブリへの間接参照 '&lt;assemblyname&gt;', を含む'&lt;typename&gt;'
コンス トラクターは、アセンブリへの間接参照 '\<assemblyname >'、含む\<typename >。 ファイル参照を追加\<filename > をプロジェクトにします。  
  
 クラス、構造体、インターフェイス、列挙型、デリゲートなどの型が式で使用されていますが、アセンブリには、型を定義するアセンブリへのプロジェクト参照がありません。  
  
 **エラー ID:** BC31528  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロジェクトのプロパティに、使用する型が定義されているアセンブリを含むファイルへの参照を追加します。
