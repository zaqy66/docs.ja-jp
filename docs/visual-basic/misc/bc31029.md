# <a name="method-ltmethodnamegt-cannot-handle-event-lteventnamegt-because-they-do-not-have-the-same-signature"></a>メソッド '&lt;methodname&gt;'イベントを処理できない'&lt;eventname&gt;' と同じシグネチャがあるないためです
指定されたイベント ハンドラーのパラメーターまたはパラメーターの型が、指定されたイベントに対して定義されたものと一致しません  
  
 **エラー ID:** BC31029  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   パラメーター リストが一致するように、イベントまたはイベント ハンドラーを再定義します。
