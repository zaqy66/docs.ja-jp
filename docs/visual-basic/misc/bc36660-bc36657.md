<span data-ttu-id="f0e55-103">メソッド内の型パラメーターのデータ型を '\<methodname >' が同じ型に変換されないために、これらの引数から推論することはできません。</span><span class="sxs-lookup"><span data-stu-id="f0e55-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="f0e55-104">データ型を明示的に指定すると、このエラーが修正される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="f0e55-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="f0e55-105">ジェネリック プロシージャの呼び出しを評価するときに、型の推定を使用して、1 つ以上の型パラメーターに対して 1 つ以上のデータ型を決定しようとしました。</span><span class="sxs-lookup"><span data-stu-id="f0e55-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="f0e55-106">コンパイラは、すべての引数の制約を満たすデータ型を見つけることができませんでした。</span><span class="sxs-lookup"><span data-stu-id="f0e55-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="f0e55-107">そのため、このエラーが報告されました。</span><span class="sxs-lookup"><span data-stu-id="f0e55-107">Therefore, it reported this error.</span></span>

>  <span data-ttu-id="f0e55-108">引数の指定がオプションではない場合 (たとえば、クエリ式内のクエリ演算子など)、エラー メッセージの 2 つ目の文は表示されません。</span><span class="sxs-lookup"><span data-stu-id="f0e55-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="f0e55-109">次のコードはエラーを示しています。</span><span class="sxs-lookup"><span data-stu-id="f0e55-109">The following code demonstrates the error.</span></span>

```vb  
    Sub Main()  
  
        '' Not valid. Integer and Date do not convert to the same type.

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
End Module
```  

## <a name="to-correct-this-error"></a><span data-ttu-id="f0e55-111">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="f0e55-111">To correct this error</span></span>

