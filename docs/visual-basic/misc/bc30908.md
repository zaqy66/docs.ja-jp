<span data-ttu-id="9a5ec-103">クラスまたはインターフェイスの定義に、その内に入れ子にされた型を指定する [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) が含まれています。</span><span class="sxs-lookup"><span data-stu-id="9a5ec-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>

```  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class

```
