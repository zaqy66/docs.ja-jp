# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<typename >' 'mustoverride' と宣言されているメソッドが含まれているため、'mustinherit ' としてを宣言する必要があります
`MustOverride` メンバーを含む型は、 `MustInherit`として宣言する必要があります。  
  
 **エラー ID:** BC31411  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型に `MustInherit` 修飾子を追加します。
