# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-type-lttypenamegt-is-declared-mustinherit-and-cannot-be-created"></a>ラムダ式に変換できません '&lt;typename&gt;' ため、型 '&lt;typename&gt;' が 'mustinherit ' として宣言されているし、作成することはできません
ステートメントがラムダ式を型に変換しようとしていますが、その型は基底クラスにすることのみができ、インスタンスの作成には使用できません。  
  
 **エラー ID:** BC36626
