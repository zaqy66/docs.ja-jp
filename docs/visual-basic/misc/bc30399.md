# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>メソッドでは、'MyBase' を使用できません '\<methodname >' 'mustoverride' と宣言されているため、
`MyBase` が宣言されているメソッドと共に `MustOverride`を使用しようとしました。  
  
 **エラー ID:** BC30399  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `MustOverride` 宣言を削除します。
