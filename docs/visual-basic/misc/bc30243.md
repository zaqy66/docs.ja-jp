# <a name="ltmodifiergt-is-not-valid-on-an-event-declaration"></a>'&lt;修飾子&gt;' イベントの宣言では無効です
`Event` ステートメントに `ReadOnly`などの無効なキーワードが含まれています。  
  
 **エラー ID:** BC30243  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Event` ステートメントから使用できないキーワードを削除します。
