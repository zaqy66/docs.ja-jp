# <a name="ltnamegt-cannot-expose-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-lttypegt"></a>'&lt;名前&gt;'は、基になるデリゲート型を公開できません'&lt;delegatetype&gt;'経由でプロジェクトの外側に実装するイベントの&lt;指定子&gt;'&lt;型。&gt;'
このコードは、パブリック クラスの外部でプライベート型を公開しています。  
  
 **エラー ID:** BC30915  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型 `Public` を宣言します。  
  
     または  
  
-   別の型を使用します。
