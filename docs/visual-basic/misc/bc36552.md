## <a name="example"></a><span data-ttu-id="3fa56-110">例</span><span class="sxs-lookup"><span data-stu-id="3fa56-110">Example</span></span>

```  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="3fa56-112">拡張メソッドが次のように呼び出される場合、そのメソッドのパラメーター `str` は `greeting`( `String` を呼び出す `Print`のインスタンス) にバインドされます。</span><span class="sxs-lookup"><span data-stu-id="3fa56-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="3fa56-113">コンパイラは、拡張メソッド `greeting` の引数として `Print`を使用します。</span><span class="sxs-lookup"><span data-stu-id="3fa56-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>
