## <a name="example"></a><span data-ttu-id="1eadd-109">例</span><span class="sxs-lookup"><span data-stu-id="1eadd-109">Example</span></span>  
 <span data-ttu-id="1eadd-110">次のクラスには、整数引数を必要とする既定のプロパティ `defaultProp`が含まれています。</span><span class="sxs-lookup"><span data-stu-id="1eadd-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

