# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-its-container-ltcontainernamegt-is-not-declared-public"></a>型 '&lt;typename&gt;' ため、属性として使用することはできません、コンテナー '&lt;containername&gt;' 'Public' と宣言されていません
この属性が定義されているクラスまたはモジュールが `Public` 修飾子を使用して宣言されていません。 アクセス修飾子が指定されていないクラスとモジュールは、既定で `Friend` として宣言されます。  
  
 **エラー ID:** BC31517  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  この属性が定義されているクラスまたはモジュールに `Public` 修飾子を追加します。
