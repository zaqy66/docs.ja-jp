# <a name="myclass-cannot-be-used-outside-of-a-class"></a>'MyClass' をクラスの外で使用することはできません
クラス構造の外部で `MyClass` キーワードを使用しました。  
  
 **エラー ID:** BC30470  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `MyClass` 宣言をクラス構造の中に移動します。
