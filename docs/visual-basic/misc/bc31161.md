# <a name="expected-closing---gt-for-xml-comment"></a>終わり '--&gt;' の XML コメント
XML コメント リテラルの終わりに必要な '-->' 文字がありません。  
  
 **エラー ID:** BC31161  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント リテラルの終わりに必要な '-->' 文字を追加します。
