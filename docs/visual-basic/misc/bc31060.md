# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>'\<propertyname >' を暗黙的に定義'\<membername >'、同じ名前のメンバーと競合する\<型 > '\<typename >'
型メンバーの名前が、プロパティに対して暗黙的に作成されたメンバーの名前と競合します。 プロパティは、定義されているメンバーと競合する可能性がある複数の暗黙的な変数を暗黙的に作成します。  
  
 **エラー ID:** BC31060  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   明示的に宣言したメンバーの名前を変更して、名前の競合を解決します。
