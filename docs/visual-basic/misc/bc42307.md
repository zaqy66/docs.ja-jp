# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>XML コメント パラメーター '\<パラメーター >' に対応するパラメーターと一致しません\<キーワード > ステートメント
XML コメント パラメーター\<パラメーター > に対応するパラメーターと一致しません\<キーワード > ステートメント。 XML コメントは無視されます。  
  
 XML コメント タグが、メソッドの一部の (すべてのではない) パラメーターに対して指定されています。  
  
 **エラー ID:** BC42307  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント タグを、メソッドのすべてのパラメーターに指定します。
