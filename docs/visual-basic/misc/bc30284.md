# <a name="type1-typename-cannot-be-declared-overrides-because-it-does-not-override-a-type1-in-a-base-type2"></a>\<type1 > '\<typename >' オーバーライドしないため、'Overrides' を宣言することはできません、 \<type1 > ベース\<type2 >
`Sub`、 `Function`、 `Property` のいずれかのステートメントに `Overrides` を指定していますが、同じ名前の型が基底クラスに存在しません。  
  
 **エラー ID:** BC30284  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  型名のスペルが正しいことを確認します。  
  
2.  余分な `Overrides` キーワードを削除します。
