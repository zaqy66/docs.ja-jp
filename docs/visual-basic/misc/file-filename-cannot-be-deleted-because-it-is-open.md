# <a name="file-ltfilenamegt-cannot-be-deleted-because-it-is-open"></a>ファイル '&lt;filename&gt;' が開いているために削除できません
開いているファイルを削除しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを閉じてから削除します。
