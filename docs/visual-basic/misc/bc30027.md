# <a name="end-function-expected"></a>'End Function' が必要です
`End` ステートメントは、終了するブロックと一致する必要があります。 `Function` プロシージャの定義が正しく終了していません。  
  
 **エラー ID:** BC30027  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End Function` ステートメントをプロシージャの最後に追加します。
