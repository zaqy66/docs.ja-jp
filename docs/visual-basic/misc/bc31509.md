# <a name="unable-to-open-resource-file-filename-error"></a>リソース ファイルを開けません '\<filename >':\<エラー >。
指定したリソース ファイルを開くときにエラーが発生しました。  
  
 **エラー ID:** BC31509  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  指定した名前と場所が正しいことを確認します。  
  
2.  ファイルが破損していないことを確認します。
