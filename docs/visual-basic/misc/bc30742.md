# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>値 '&lt;valuename1&gt;'に変換できません'&lt;valuename2&gt;'
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30742  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   キャストされている型を確認します。
