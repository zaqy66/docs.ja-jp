# <a name="persistabletype"></a>\<persistableType >
すべての永続型を指定します。  
  
 \<system.ServiceModel >  
\<comContracts>  
\<comContract>  
  
## <a name="syntax"></a>構文  
  
```xml  
<comContracts>
  <comContract>
    <persistableTypes>
      <persistableType id="String"
                       name="String">
      </persistableType>
    </persistableTypes>
  </comContract>
</comContracts>
```  
  
## <a name="type"></a>型  
 `Type`  
  
## <a name="attributes-and-elements"></a>属性および要素  
 以降のセクションでは、属性、子要素、および親要素について説明します。  
  
### <a name="attributes"></a>属性  
  
|属性|説明|  
|---------------|-----------------|  
|id|永続型の一意の ID を指定する文字列を含む必須属性。|  
|name|永続型の名前を指定する文字列を含む省略可能な属性。|  
  
### <a name="child-elements"></a>子要素  
 なし  
  
### <a name="parent-elements"></a>親要素  
  
|要素|説明|  
|-------------|-----------------|  
|[\<persistableTypes>](../../../../../docs/framework/configure-apps/file-schema/wcf/persistabletypes.md)|`persistableType` 要素のコレクション。|
