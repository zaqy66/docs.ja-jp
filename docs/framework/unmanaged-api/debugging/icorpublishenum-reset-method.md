# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset メソッド
列挙体の先頭のカーソルを移動します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Reset ();  
```  
  
#### <a name="parameters"></a>パラメーター  
 `Reset` パラメーターはありません。
