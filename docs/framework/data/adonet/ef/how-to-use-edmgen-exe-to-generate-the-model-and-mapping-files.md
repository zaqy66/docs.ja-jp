### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="037e0-116">EdmGen.exe を使用して Visual Basic プロジェクト用の School モデルを生成するには</span><span class="sxs-lookup"><span data-stu-id="037e0-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="037e0-119">コマンド プロンプトで、次のコマンド (改行なし) を実行します。</span><span class="sxs-lookup"><span data-stu-id="037e0-119">At the command prompt, execute the following command without line breaks:</span></span>

