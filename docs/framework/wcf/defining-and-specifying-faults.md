### <a name="defining-error-conditions-that-clients-should-know-about"></a><span data-ttu-id="0b7ae-115">クライアントに通知する必要があるエラー状態の定義</span><span class="sxs-lookup"><span data-stu-id="0b7ae-115">Defining Error Conditions That Clients Should Know About</span></span>
